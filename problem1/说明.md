# 问题分析

## 基本假设

**假设 1（样本独立同分布）**：A 的每一行为一个独立的监测样本，各行之间互不相关且来自同一分布。这是最小二乘估计统计性质（无偏性、有效性）的基础，也是后文 Durbin-Watson 独立性检验的前提。

**假设 2（线性生成机制）**：B 由 A 的各维度线性叠加并叠加噪声生成，即存在固定权重向量 $w\in\mathbb{R}^{99}$ 与偏置 $b$，使

$$B = Aw + b\mathbf{1} + \varepsilon$$

依据：探索性分析中 B 与 A 行和的相关系数达 0.874，普通线性回归 $R^2$ 已达 0.989。该假设的充分性将在"分析误差来源"一节通过二次项对照实验加以验证。

**假设 3（零均值、同方差、对称的轻尾噪声）**：$\varepsilon_i$ 独立同分布，$E[\varepsilon_i]=0$，$\mathrm{Var}(\varepsilon_i)=\sigma^2$ 为常数。

依据：这是高斯-马尔可夫定理的前提，保证最小二乘估计为最优线性无偏估计（BLUE）。其具体分布形态将由残差检验识别（结果表明噪声近似服从均匀分布，详见误差分析）。

**假设 4（无完全多重共线性）**：增广矩阵 $X=[A,\ \mathbf{1}]$ 列满秩，即 $\mathrm{rank}(X)=100$。

依据：计算得 A 各列间样本相关系数绝对值最大仅 0.035，$X^TX$ 条件数为 $8.21\times10^3$，数值性态良好，$(X^TX)^{-1}$ 存在且稳定。

**假设 5（量纲一致，无需额外预处理）**：A 的全部元素均位于 $[0,1]$ 区间，各维度量纲一致，可直接参与线性组合；B 作为目标变量保持原始尺度，以便误差指标具有可解释的实际意义。

## 问题1的分析

问题 1 要求对数据 A（$10000\times 99$）作某种变换，使变换结果尽可能接近数据 B（$10000\times 1$），并计算与分析误差。其数学本质是：寻找一个映射 $f:\mathbb{R}^{99}\to\mathbb{R}$，使 $f(A)\approx B$。

对数据的初步观察：

- B 的均值为 404.48、标准差为 16.72，而 A 的各列均近似服从 $U(0,1)$；
- B 与 A 的行和相关系数高达 0.874，提示 B 可能是 A 各维度的加权和；
- 预实验中普通最小二乘回归的 $R^2$ 已达 0.989，说明线性结构占绝对主导。

据此采用"**从简到繁、假设检验驱动**"的建模路线：先建立最简单的线性变换模型并用最小二乘法求解析解；再通过二次项对照实验检验是否存在被忽略的非线性结构（模型偏差）；最后对残差做系统的统计检验，识别误差的真实来源（噪声特性）。

# 模型的建立与求解

## 模型的建立

**1. 模型形式**

对第 $i$ 个样本（$i=1,\dots,10000$），设

$$B_i = \sum_{j=1}^{99} w_j A_{ij} + b + \varepsilon_i$$

矩阵形式为 $B = X\beta + \varepsilon$，其中 $X=[A,\ \mathbf{1}]\in\mathbb{R}^{10000\times 100}$，$\beta=(w^T,\ b)^T$。

**2. 参数估计：最小二乘法**

取目标函数为残差平方和：

$$\min_{\beta}\ J(\beta)=\|X\beta-B\|^2$$

$J$ 关于 $\beta$ 为凸二次函数，全局最优唯一。令 $\nabla_\beta J = 2X^T(X\beta-B)=0$，得**正规方程**：

$$X^TX\hat{\beta}=X^TB$$

由假设 4，$X^TX$ 可逆，得闭式解：

$$\boxed{\ \hat{\beta}=(X^TX)^{-1}X^TB\ }$$

**3. 方法选择的论证**（为何不直接"求逆"或"用神经网络"）

- A 为 $10000\times 99$ 非方阵，$A^{-1}$ 不存在；该方程组含 10000 个方程、100 个未知数，属**超定方程组**。由于 B 含噪声，任意取 100 行求逆所得的解会过拟合所选行的噪声，且随选行不同而变化，不具备统计意义；
- 由假设 3 与高斯-马尔可夫定理，最小二乘估计是所有线性无偏估计中方差最小者（BLUE）；
- 该问题为凸优化问题，闭式解精确、可复现、无超参数；迭代式方法（如梯度下降训练线性层）收敛目标与本解相同，但只给出近似解且引入不必要的调参与随机性，故不采用。

**4. 求解算法**

实际计算采用基于奇异值分解（SVD）的最小二乘算法（`numpy.linalg.lstsq`），避免显式形成与求逆 $X^TX$，数值稳定性更好。时间复杂度 $O(np^2)$（$n=10^4,\ p=100$），实测毫秒级完成。

## 模型的求解与结果

**1. 拟合结果**

| 指标 | 数值 |
|---|---|
| 决定系数 $R^2$ | **0.9891** |
| 调整 $R^2$ | 0.9890 |
| RMSE | **1.7434** |
| MAE | 1.5087 |
| 噪声方差估计 $\hat\sigma^2$（MSE） | 3.0395 |
| 偏置 $\hat b$ | 151.6440 |

权重 $\hat w$ 的分布：均值 5.1024，标准差 2.8109，取值范围 $[0.1285,\ 9.7661]$。$\hat w_j$ 可解释为"第 $j$ 个监测维度对目标量 B 的贡献权重"。

> 📊 【插图 1】权重向量 $\hat w$ 柱状图（99 根柱），突出权重最大/最小的前 10 个维度。

**2. 结果解读**

B 的总方差为 $\mathrm{Var}(B)\approx 279.6$，其中由线性变换 $Aw+b$ 解释的部分占 **98.91%**，剩余约 **1.09%**（$\hat\sigma^2\approx 3.04$）为不可约的噪声方差。模型以 100 个参数拟合 10000 个样本，调整 $R^2$ 与 $R^2$ 几乎一致，说明无过拟合、参数规模合理。

## 分析误差来源

总误差可分解为**模型偏差**与**数据噪声**两部分，下面分别定量检验。

**1. 模型偏差检验（线性假设是否充分）**

构造扩展模型，加入全部二次项 $A_{ij}^2$（特征维度由 100 增至 199）重新拟合：

| 模型 | $R^2$ |
|---|---|
| 线性模型 | 0.989127 |
| 线性 + 二次项模型 | 0.989235 |

$R^2$ 仅提升约 $1\times10^{-4}$，非线性项几乎不贡献解释力。**结论：模型偏差可忽略，线性假设充分**，误差几乎全部来自噪声。

**2. 残差统计检验（噪声特性识别）**

记残差 $\hat\varepsilon = B - X\hat\beta$，检验结果汇总：

| 检验项 | 统计量 | 判定基准 | 结论 |
|---|---|---|---|
| 独立性（Durbin-Watson） | 1.9908 | ≈2 为无自相关 | ✅ 残差序列无自相关 |
| 与拟合值相关性 | $-5.8\times10^{-14}$ | ≈0 | ✅ 正交（OLS 理论性质，数值验证） |
| 对称性（偏度） | 0.0010 | ≈0 为对称 | ✅ 分布高度对称 |
| 峰度（超额） | **−1.1799** | 正态为 0，**均匀分布为 −1.2** | ⚠ 显著偏离正态，**极度接近均匀分布** |
| 正态性（Jarque-Bera） | 580.10 | 临界值 $\chi^2_{0.05}(2)=5.99$ | ❌ 拒绝正态假设 |
| 正态性（KS） | 0.0589 | 临界值 $1.36/\sqrt n=0.0136$ | ❌ 拒绝正态假设 |
| 同方差（Breusch-Pagan） | 130.51 | 临界值 $\chi^2_{0.05}(99)=123.23$ | ⚠ 临界水平轻微拒绝，考虑其由非正态性放大（BP 对峰度敏感），结合残差-拟合值散点图无明显漏斗形态，判定方差基本恒定 |

**3. 噪声分布形态的进一步识别**

残差的超额峰度 −1.18 与均匀分布的理论值 −1.2 几乎一致，提示 $\varepsilon\sim U(-a,a)$。由 $\mathrm{Var}(\varepsilon)=a^2/3$ 估计得 $\hat a=\sqrt{3\hat\sigma^2}\approx 3.02$。进一步观察：

- 残差实际范围为 $[-3.60,\ 3.50]$，约 98.1% 的残差落在 $(-3.02,\ 3.02)$ 内，仅 188/10000 略超出；
- 对 $U(-3.02,\ 3.02)$ 的卡方拟合优度检验统计量为 21.75（临界值 $\chi^2_{0.05}(9)=16.92$），轻微拒绝严格的均匀分布。

**结论：噪声为零均值、对称、有界、近似均匀分布的轻尾噪声，半宽约 3.0，可能存在少量越界样本（约占 1.9%）。** 这一发现具有实际意义：矿山监测中的量化误差、传感器截断误差通常服从均匀分布而非正态分布，与识别结果吻合。

**4. 误差的量化分解**

$$\underbrace{\mathrm{Var}(B)}_{279.6} = \underbrace{\mathrm{Var}(A\hat w+\hat b)}_{\text{模型解释 }98.91\%} + \underbrace{\hat\sigma^2}_{\text{噪声 }1.09\%},\qquad \text{模型偏差}\approx 0$$

# 小结

针对问题 1，建立了线性变换模型 $B=Aw+b\mathbf{1}+\varepsilon$，基于最小二乘法导出闭式解 $\hat\beta=(X^TX)^{-1}X^TB$ 并采用 SVD 算法求解。模型拟合优度 $R^2=0.9891$、RMSE=1.7434。误差来源分析表明：

1. **模型偏差可忽略**——加入二次项后 $R^2$ 仅提升 $10^{-4}$ 量级；
2. **误差几乎全部来自数据噪声**（占总方差约 1.09%），噪声经识别为零均值、对称、近似均匀分布 $U(-3,3)$ 的有界轻尾噪声，与监测设备量化误差的物理机制一致；
3. 残差通过独立性与（基本）同方差检验，验证了最小二乘框架的适用性。

该结果为后续问题 3 的 X–Y 关系建模提供了方法基础与噪声认识。
